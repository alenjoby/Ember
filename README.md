# Ember: A Calm, Zero-Chrome Digital Sanctuary

Ember is a digital space designed for quiet reflection and emotional sharing. Anonymous thoughts transform into floating lanterns within an atmospheric, spatial UI.

## Features

- **Organic Scatter Layout**: Avoids rigid grids with an atmospheric depth simulation.
- **Interactive Response Drawer**: Support for notes, stickers, voice messages, and drawings.
- **AI Fallback System**: Triggers a comforting voice and audio-reactive glow if no human interaction occurs.
- **Spatial Interactions**: Lanterns react to mouse movements with subtle repulsion.

## Tech Stack

- **Frontend**: React, Tailwind CSS v4
- **Animation**: Motion (formerly Framer Motion)
- **Icons**: Lucide React
- **UI Components**: Radix UI, Shadcn UI (customized)
- **Deployment**: Vite
